# Fitpoise - GridsomeJS & StrapiJS fitness web app

Fitpoise uses Gridsome as the front end and Strapi as the  
## Installation

Open two command prompt windows and type the following:

**CMD Prompt 1**
```bash
cd ./fitpoise-api/
yarn install
yarn develop
```
**CMD Prompt 2**
```bash
cd ./fitpoise-frontend/
npm install
gridsome develop
```
